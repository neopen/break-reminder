## 需扩展的功能

### 🔴 高优先级（用户体验提升）

| 功能              | 说明                                               | 理由                                         |
| :---------------- | :------------------------------------------------- | :------------------------------------------- |
| **通知栏提醒**    | 即使浏览器在后台或最小化，也能通过系统通知提醒用户 | 当前弹框只在页面打开时有效，关闭浏览器就失效 |
| **每日统计/打卡** | 记录用户完成了多少次活动提醒，形成打卡记录         | 增加用户粘性，让用户有成就感                 |
| **休息动作引导**  | 弹框内展示简单的拉伸动画或 GIF                     | 不只是提醒，还教用户怎么做                   |
| **白名单模式**    | 用户可以设置免打扰时间段（如午休12:00-13:00）      | 更符合实际工作场景                           |

### 🟡 中优先级（功能增强）

| 功能             | 说明                                                         | 理由                     |
| :--------------- | :----------------------------------------------------------- | :----------------------- |
| **多组时间段**   | 支持设置多组不同的时间段和频率（如上午9-12点/30分钟，下午2-6点/45分钟） | 更灵活，适应不同工作节奏 |
| **提醒历史记录** | 记录每次提醒的时间和用户的响应（活动了/跳过）                | 帮助用户复盘习惯养成     |
| **自定义提示语** | 用户可以修改弹框提示文字                                     | 个性化需求               |
| **主题切换**     | 深色/浅色/多套配色主题                                       | 提升视觉体验             |
| **导出配置**     | 支持导出/导入设置，方便换设备迁移                            | 实用性                   |

### 🟢 低优先级（锦上添花）

| 功能       | 说明                                     |
| :--------- | :--------------------------------------- |
| 音效选择   | 多种提示音可选（鸟叫、流水、经典闹钟等） |
| 振动提醒   | 移动端支持振动（需用户授权）             |
| 数据同步   | 通过云端同步多设备配置                   |
| 桌面小部件 | 桌面显示下次提醒时间的 Widget            |
| 番茄钟模式 | 结合番茄工作法（25分钟专注 + 5分钟休息） |



### 简化配置，增加「快速模板」

| :------------ | :---------------------------------- |
